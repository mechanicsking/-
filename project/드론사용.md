```
mkdir -p ~/tello_ros_ws/src
cd ~/tello_ros_ws/src
git clone https://github.com/TIERS/tello-ros2-gazebo.git
cd ..
source /opt/ros/galactic/setup.bash
colcon build
```
위에를 한줄씩 추가한다음 (colcon build 안될것임)   
```
gedit ~/.bashrc
```
bashrc를 열어 다음을 추가하여 tello_ros_ws를 source 하는 걸 추가해준다. 
```
source $HOME/tello_ros_ws/install/setup.bash
```
다시 터미널에게 돌아와서 소스해준다. 
```
source gedit ~/.bashrc
```
code .을  통해 tello_ros_gazebo파일을 열고 lauch파일에서 simple_lauch.py를 연다음 다음줄을 앞에 다 #처리한다. 
```python
        # Joystick driver, generates /namespace/joy messages
        #Node(package='joy', executable='joy_node', output='screen',
         #    namespace=ns),

        # Joystick controller, generates /namespace/cmd_vel messages
        #Node(package='tello_driver', executable='tello_joy_main', output='screen',
         #    namespace=ns),
```
저장한다음 다시 나와서 터미널에서 tello_ros_ws에서 colcon build를 해주는 다음의 명령어로 진행한다.   
```
colcon build --packages-select tello_gazebo
```
```
ros2 launch tello_gazebo simple_launch.py
```
실행 후 다른 터미널에서 
```
ros2 service call /drone1/tello_action tello_msgs/TelloAction "{cmd: 'takeoff'}"
```
실행하며 드론이 뜬다. 
```
ros2 run teleop_twist_keyboard teleop_twist_keyboard --ros-args -r cmd_vel:=/drone1/cmd_vel
```
를 실행하면 키보드로 드론을 움직일 수 있다.
