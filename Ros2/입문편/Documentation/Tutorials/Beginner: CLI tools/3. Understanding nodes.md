## Background
### 1. The Ros2 graph
앞으로 남은 몇가지 튜토리얼에서는 “ROS (2) graph"라 불리는 ros2의 core concept들을 배울 것이다. ros graph는 데이터를 동시에 처리하는 ros2요소의 네트워크이다. 모든 실행 파일과 이들 간의 연결을 모두 매핑하여 시각화하면 이를 모두 포괄한다.
